# Claude Code Integration Files

This folder contains the Claude Code integration for MacroQuest.

**For installation and usage instructions, see the main documentation:**
[Claude Code Integration Guide](../../main/claude-code-integration.md)

## Files

| File | Purpose |
|------|---------|
| `mq-stub.md` | Template to copy to `~/.claude/commands/mq.md` |
| `mq-full.md` | Coordinator logic |
| `MacroQuest-Researcher-Full.md` | Research agent instructions |
| `MacroQuest-Expert-Full.md` | Code expert agent instructions |

## Quick Install

1. Copy `mq-stub.md` to `~/.claude/commands/mq.md`
2. Edit the paths in your copy
3. Use `/mq` in Claude Code
