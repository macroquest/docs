# `MacroQuest`

Creates an object related to MacroQuest information.

## Forms

* [_macroquest_](../data-types/datatype-macroquest.md) **MacroQuest**

## Examples

```
/echo ${MacroQuest.LastTell}
```

Returns the name of the last person that sent you a tell.
