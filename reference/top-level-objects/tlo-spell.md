# `Spell`

Object used to return information on a spell by name or by ID.

## Forms

[_spell_][spell] **Spell**[_N_]

:   Find spell by ID

[_spell_][spell] **Spell**[_name_]

:   Find spell by name

## Examples

```
/echo ${Spell[Splurt].ID}
```

Will return 1620

```
/echo ${Spell[1620].Duration}
```

Will return 16 (ie. 16 ticks)
