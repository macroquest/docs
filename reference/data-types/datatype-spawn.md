## Methods

| Name           | Action                 |
| -------------- | ---------------------- |
| **DoAssist**   | assists the spawn      |
| **DoFace**     | Faces target           |
| **DoTarget**   | Targets the spawn      |
| **LeftClick**  | left clicks the spawn  |
| **RightClick** | Right clicks the spawn |
