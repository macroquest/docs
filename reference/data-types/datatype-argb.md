# `argb`

Represents a color

## Members

### [int][int] `A`

:   Alpha channel value.

### [int][int] `R`

:   Red color value.

### [int][int] `G`

:   Green color value.

### [int][int] `B`

:   Blue color value.

### [int][int] `Int`

:   The integer formed by the ARGB.

### [string][string] To String

:   The hex value of the integer formed by the ARGB.
