# `argb`

Represents a color

## Members

[_int_][int] **A**

:   Alpha channel value.

[_int_][int] **R**

:   Red color value.

[_int_][int] **G**

:   Green color value.

[_int_][int] **B**

:   Blue color value.

[_int_][int] **Int**

:   The integer formed by the ARGB.

[_string_][string] **(To String)**

:   The hex value of the integer formed by the ARGB.
