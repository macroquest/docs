# `fellowshipmember`

Contains all the data related to fellowship members

## Members

### [class][class] `Class`

:   Member's class

### [ticks][ticks] `LastOn`

:   How long since member was last online

### [int][int] `Level`

:   Member's level

### [string][string] `Name`

:   Member's name

### [bool][bool] `Sharing`

:   TRUE if member has exp sharing enabled

### [zone][zone] `Zone`

:   Zone information for the member's zone

### [string][string] `To String`

:   player name
