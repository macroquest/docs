# /mqpause

## Syntax

```eqcommand
/mqpause [on|off] | chat [on|off]
```

## Description

Pauses/resumes a macro. Not using a parameter will toggle pausing on/off.

## Parameters

- **on|off** - Pauses or resumes a macro
- **chat [on|off]** - controls whether macro will pause while in chat mode
